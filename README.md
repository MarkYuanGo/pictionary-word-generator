## 🎨 What is PictionaryHub?

PictionaryHub brings the classic "sketch and guess" board game into the digital age. Whether you are hosting a **Karaoke intermission**, a **family game night**, or a **virtual team-building event**, our platform provides the canvas and the inspiration.

### Why choose PictionaryHub?

  * **Zero Installation:** Play directly in your browser on desktop, tablet, or mobile.
  * **Massive Word Bank:** From "Easy" lists for kids to "Insane" challenges for pros.
  * **Vibe-Ready:** Perfect for streaming on Discord, Zoom, or big-screen TVs.
  * **Touch Optimized:** Works beautifully with mouse, stylus, or your finger.

## 🛠 Tech Stack (Built with Passion)

PictionaryHub is a high-performance web application built using modern web technologies to ensure low latency and smooth drawing:

  * **Frontend:** Next.js & Tailwind CSS
  * **Backend:** Java (Spring Boot) & Python AI Services
  * **Infrastructure:** Scalable cloud deployment for global access
